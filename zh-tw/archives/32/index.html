[Linux]ubuntu食用指北（重製版）

  • 3448字數

本教程旨在給讀者介紹一個優秀的系統。

其實自從用了macOS之後就再沒用過ubuntu的桌面版了，主要都是伺服器版，畢竟這個OS不用命令列(CLI)簡直就是暴殄天物，還不如用macOS呢（macOS第一！（破音）），所以這次重置（己亥年六月十三完稿，啜著Liption紅茶）主要增加了很多CLI介紹。

常用命令

ls

全稱我忘記了(list?)，這大概是我最常用的一條命令了，用途便是列舉目錄下檔案：

options（常用）:
-h  aka.human，人性化顯示（針對很多命令都有類似效果
-l  列出每個檔案（夾）的許可權、使用者組和所有者
-s  顯示檔案大小，資料夾則是0B（macOS下），單位好像就是B

e.g:
ls -hs   以人性化的方式顯示檔案大小
ls -l    就是-l引數的用途
ls /path 列出path目錄下檔案（夾）

Windows下面沒有ls這個命令，以dir取代之。

rm

rm是remove嘛？

這個命令比較危險⚠️，刪了就刪了，沒有後悔藥可吃（吧），甚至許可權足夠可刪除根目錄。

rm index.html  刪除index.html
rm -r /path    刪除目錄（使用-r引數）

mv

mv就是move咯。

如其名字，兼具移動和重新命名兩大功能（大概改名字就是原地移動吧）。

mv /path/index.html /path2/             將index.html移到/path2
mv /path/index.html /path2/index2.html  將index.html移到/path2，並改名為index2.html
mv /path /path2                         將path更名為path2
注意⚠️，同名檔案會直接覆蓋

pwd

用於顯示當前路徑，順便提下相對路徑和絕對路徑：

絕對路徑就是檔案的真正存在的路徑，是指從硬碟的根目錄(碟符)開始，進行一級級目錄指向檔案。

相對路徑就是以當前檔案為基準進行一級級目錄指向被引用的資原始檔。

另外.和..分別代表當前目錄和上級目錄。

還有執行一個當前路徑下的程式、指令碼等仍需指明其相對或絕對路徑，比如在/path下有個指令碼run.sh，而你位於這個路徑，想執行指令碼，直接敲上run.sh會提示找不到檔案，需要這樣./run.sh或者/path/run.sh。

>

>嚴格來說好像不算命令，在我學習競賽的時候某著名教程上將其作為管道解，用於重定向輸出，一般有些輸出是直接輸出在螢幕上的，但也可以通過>重定向到檔案。

比如：ls > a.txt就會將ls的輸出轉到a.txt裏面去。

echo

回顯輸入，echo hello系統輸出hello。

但是其實這個命令我用了建立檔案的：echo > index.html。

curl

可以下載檔案，訪問網頁（命令列瀏覽器，直接得到原始碼233）。

curl https://4o5.xyz               這樣會得到我網站主頁的原始碼，直接輸出到螢幕。
curl https://4o5.xyz > index.html  得到程式碼再儲存到index.html中

這個命令還有更多用法，不過我也就只用來幹這件事了。

wget

wget是一個更純粹的命令列下載器。

wget https://4o5.xyz/1.tar.gz     下載1.tar.gz
wget -c https://4o5.xyz/1.tar.gz  斷點續傳下載1.tar.gz

ssh&sftp

一般萌新連線伺服器是用GUI程式的，那個我一開始也是嚴重依賴，後來發現macOS自帶了這個命令，這類客戶端就直接被我解除安裝了（再後來發現win10 1803更新之後也有很多linux命令了(tar,ssh,sftp,wget,curl…)）。

ssh usr@ip OR domain name
sftp usr@ip OR domain name

ssh連上去了就沒啥好說了。

sftp這裏介紹一下，我也是比較晚才知道有這個命令的存在，天知道以前怎麼上傳檔案的。
sftp連上去之後可以使用ls,cd,put,get,exit等命令來操作遠端主機。
ls,cd,exit就不說了。
get是將遠端主機檔案拉回本地，資料夾加上-r引數。
put則是相反。
在sftp裏面同時也可以操作本地主機，在ls,cd前各加上一個l就行了，像這樣：lls,lcd。

tar

tar很強大的打包軟體，可與gzip等壓縮軟體聯動。

具體單個引數網路上都有了我就不重複造輪子了。

tar -czvf 1.tar.gz /path  將/path打包並用gzip壓縮
tar -xzvf 1.tar.gz        將1.tar.gz解包

其他

mkdir   建立目錄，加上-p引數可一次性建立多級目錄
df -hl  檢視掛載點可用空間，說白了就是看硬碟餘量
top     命令列版taskmgr
free -h 人性化檢視可用記憶體
shutdown -P  一分鐘後關機
reboot  重啟
rm -rf /  跑路！

小提示：

按鍵盤上的⬆️和⬇️可以得到歷史輸入記錄。

Tab鍵可以補全命令

ctrl+c或ctrl+z可以打斷命令執行。

以下是舊口胡版

版本選擇＆下載

本人選ubuntu18.04.1 LTS。

前往https://ubuntu.com下載，國區可前往https://cn.ubuntu.com。

安裝

安裝需要一個U盤或光碟。

對於新版本的ubuntu，不要相信所謂百度的軟碟通之類的軟體，引導不了的。

用rufus這個軟體即可。

製作時目標系統選擇「BIOS or UEFI」。

安裝時請選擇與你當前作業系統相對應的啟動方式，「Legacy」或者「UEFI」。

這樣ubuntu的grub引導就會發現windows boot manager，日後就不會為雙系統引導發愁了。

美化

對於18.04以上版本，可以去看我以前一片文章。

其中本人所用主題原託管位置關閉，請前往Github上下載。

軟體

Java

JDK安裝(openjdk)：

$ sudo apt install openjdk-8-jdk

注意，openjdk不包括JavaFX，需要安裝openjfx：

$ sudo apt install openjfx

常見問題

某某軟體打不開，沒有許可權：右鍵那個檔案，在屬性欄允許以程式執行balabala的。
