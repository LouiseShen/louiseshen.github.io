[reCAPTCHA]typecho新增reCAPTCHA驗證碼

發現typecho的這類外掛很少，而且很難用，所以乾脆現學現賣自己整個。

具體效果看本站的登陸介面就行了，handsome快捷登陸在右上角，typecho登陸則在https://4o5.xyz/admin/login.php

概述

雖然typecho的這方面外掛少，但是其本身的設計模式確實非常ok的：

admin/login.php用來前端登陸，var/Widget/Login.php則處理登陸邏輯。

所以我們改動也就只需改這裏就行啦，記得做好備份。

reCaptcha申請

前往https://www.google.com/recaptcha/admin 申請一個key，只有這一步需要訪問google.com，其餘不再需要。程式碼已經使用recaptcha的google官方反向代理，國內正常使用。

申請的時候記得選擇reCAPTCHA v2版本，v3我沒試過不保證成功。

登陸介面整合

在admin/login.php中（這裏是相對於typecho的安裝位置）

引入js：<script src="https://www.recaptcha.net/recaptcha/api.js"></script>（不會的話就隨便放在一個

標籤的前面）

然後大概在第26行，就是<input type="password"......>的<p></p>標籤後面一行加入。

<p>
    <div class="g-recaptcha" data-sitekey="your-site-key"></div>
</p>

記得將上面的your-site-key替換成你申請的鴨。

這樣前端就有了人機身份驗證的checkbox了。

登陸驗證整合

在var/Widget/Login.php中，這裏改動很多直接替換吧（沒有後門指令碼，僅加入了recaptcha相關邏輯處理程式碼）：

<?php
if (!defined('__TYPECHO_ROOT_DIR__')) exit;
/**
 * 登入動作
 *
 * @category typecho
 * @package Widget
 * @copyright Copyright (c) 2008 Typecho team (http://www.typecho.org)
 * @license GNU General Public License 2.0
 * @version $Id$
 */

/**
 * 登入元件
 *
 * @category typecho
 * @package Widget
 * @copyright Copyright (c) 2008 Typecho team (http://www.typecho.org)
 * @license GNU General Public License 2.0
 */
class Paul_GCaptcha {
    public static $success, $failed;

    // 傳送驗證資訊
    public static function send($post_data) {
        $postdata = http_build_query($post_data);
        $options = array(
            'http' => array(
                'method' => 'POST',
                'header' => 'Content-type:application/x-www-form-urlencoded',
                'content' => $postdata,
                'timeout' => 15 * 60 // 超時時間
            )
        );
        $context = stream_context_create($options);  
        $result = file_get_contents("https://recaptcha.net/recaptcha/api/siteverify", false, $context);
        return $result;
    }

    // 判斷驗證狀況
    public static function check(){
        if($_POST["g-recaptcha-response"]){
            $data = array(
                'secret' => 'your-secret-key',
                'response' => $_POST["g-recaptcha-response"] // 接收使用者提交的驗證資料
            );

            $result = self::send($data);
            $result = json_decode($result, true);
            $result = $result["success"];

            if($result == true){
                return true; // 驗證成功
            }
            else{
                return false; // 驗證失敗
            }
        }
        else{
            return false; // 使用者沒有提交到驗證資訊
        }
    }
}

class Widget_Login extends Widget_Abstract_Users implements Widget_Interface_Do
{
    /**
     * 初始化函式
     *
     * @access public
     * @return void
     */
    public function action()
    {
        // protect
        $this->security->protect();

        /** 如果已經登入 */
        if ($this->user->hasLogin()) {
            /** 直接返回 */
            $this->response->redirect($this->options->index);
        }

        /** 初始化驗證類 */
        $validator = new Typecho_Validate();
        $validator->addRule('name', 'required', _t('請輸入使用者名稱'));
        $validator->addRule('password', 'required', _t('請輸入密碼'));

        /** 截獲驗證異常 */
        if ($error = $validator->run($this->request->from('name', 'password'))) {
            Typecho_Cookie::set('__typecho_remember_name', $this->request->name);

            /** 設定提示資訊 */
            $this->widget('Widget_Notice')->set($error);
            $this->response->goBack();
        }

        if(Paul_GCaptcha::check() == true){
            /** 開始驗證使用者 **/
            $valid = $this->user->login($this->request->name, $this->request->password,
            false, 1 == $this->request->remember ? $this->options->time + $this->options->timezone + 30*24*3600 : 0);

            /** 比對密碼 */
            if (!$valid) {
                /** 防止窮舉,休眠3秒 */
                sleep(3);

                $this->pluginHandle()->loginFail($this->user, $this->request->name,
                $this->request->password, 1 == $this->request->remember);

                Typecho_Cookie::set('__typecho_remember_name', $this->request->name);
                $this->widget('Widget_Notice')->set(_t('使用者名稱或密碼無效'), 'error');
                $this->response->goBack('?referer=' . urlencode($this->request->referer));
            }
            $this->pluginHandle()->loginSucceed($this->user, $this->request->name,
            $this->request->password, 1 == $this->request->remember);

            /** 跳轉驗證後地址 */
            if (NULL != $this->request->referer) {
                $this->response->redirect($this->request->referer);
            } else if (!$this->user->pass('contributor', true)) {
                /** 不允許普通使用者直接跳轉後臺 */
                $this->response->redirect($this->options->profileUrl);
            } else {
                $this->response->redirect($this->options->adminUrl);
            }
        }else{
            $this->widget('Widget_Notice')->set(_t('驗證失敗'),'error');
            $this->response->goBack('?referer=' . urlencode($this->request->referer));
        }
    }
}

記得將裏面的yout-secret-key換成你自己的鴨。

然後就成功了。

（這裏的驗證程式碼來自https://paugram.com/coding/add-recaptcha.html，原諒我的懶惰）

handsome快捷登陸整合

如果像我一樣使用handsome的，上述改動也會讓這裏無法登陸，所以快捷登陸這裏也整合recaptcha吧。

改動不多，這個有版權不適合放出整個檔案嘻嘻。

在usr/themes/handsome/component/headnav.php中

引入js：<script src="https://www.recaptcha.net/recaptcha/api.js"></script>

大概在第236行，也就是有一個<input type="password"......>後面一行，加入下面的程式碼：

<div class="g-recaptcha" data-sitekey="your-site-key"></div>

同理，記得將上面的your-site-key替換成你申請的鴨。
