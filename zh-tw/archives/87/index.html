QQ號被永久凍結，可dnspod解析只繫結此QQ登入咋辦？

Head Pic: original, cat ears, panties / 金髪ロリにゃんこ - pixiv

Hints

一般人都會想到先找騰訊申訴，不過永久凍結的申訴結果是這樣的：

所以一句話：小灬心灬騰灬訊。

所以說就真的沒辦法了嗎，難道只能一個個反查出解析的域名再重新解析了嗎？

不，像dnspod這類網站都有一種API的方式可以獲取賬號資訊，即不用登陸。

當然，在之前申請Let's encrypt的野卡域名的時候因為只能用dnspod外掛所以我伺服器上保留有dnspod API資訊。（ID,TOKEN格式）

利用dnspod API獲取解析列表

看了下官網API文件，我們只需要兩個操作就可以得到我們要的解析列表。

當然如果你沒有API TOKEN而且賬號被永久凍結，並且用QQ快捷登陸的話就涼涼了，沒救了。

先獲取賬號下域名的id：

curl -X POST https://dnsapi.cn/Domain.List -d "login_token=[ID,TOKEN]&format=json"

[ID,TOKEN]就是你的API key，逗號也要輸入，中括號不要輸入。

Windows10自1809版本更新後似乎內建了curl命令（pro workstation是這樣的），如果沒有可以通過mac或者linux，不礙事。

然後你的控制檯就會得到一串沒有空格沒有回車沒有格式化過的json。

推薦用工具格式化一下便於閱讀，大概像這個樣子：

{
    "status": {
        "code": "1",
        "message": "Action completed successful",
        "created_at": "2020-02-05 17:59:25"
    },
    "info": {
        "domain_total": 3,
        "all_total": 3,
        "mine_total": 3
    },
    "domains": [{
        "id": 73467724,
        "status": "enable",
        "src_flag": "DNSPOD",
        "name": "4o5.xyz",
        "records": "41"
    }]
}

重點是domains陣列裏面的id，因為後面它要按域名來查詢，所以要用到。

得到了id之後就可以查詢對應id下面的解析條目了：

curl -X POST https://dnsapi.cn/Record.List -d "login_token=[ID,TOKEN]&format=json&domain_id=[domainID]"

[domainID]就是你剛剛查詢到的id，中括號不要輸入，當然你可能不只有一個id，我有三個解析在dnspod所以這一條命令要換id發三次。
