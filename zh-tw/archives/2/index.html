如何優雅的備份伺服器

Louise

伺服器

發布於：2020年8月29日

我嫌原來寫的太水了，已經重寫了這篇文章

用github等git服務備份，不僅可以隨時看到commit修改狀況，方便回滾，還可以自動化！

（若無特殊說明，所有的命令皆在類UNIX系統下執行）

先決條件

  • github帳號
  • git
  • SSH key

1.創建SSH key

打開終端機，用下面的命令創建RSA密鑰

然後會讓你回答以下問題：

  1. Enter file in which to save the key（在哪個檔案中保存key，輸入那個檔案的路徑）
  2. Enter passphrase (empty for no passphrase)（鍵入passphrase，可以為空，但是要和下面的一致，為空就敲兩下確認鍵）
  3. Enter same passphrase again（再輸入一次）

結果大概是這樣，你會得到一個key和一個key.pub檔案（假設你保存的檔案名字叫做key）

然後將key.pub檔案裡的內容拷貝下來，再到github上，右鍵->Settings->SSH and GPG keys->New SSH key，將拷貝的內容粘貼到Key輸入框，Title由你決定，不影響。

之所以創建SSH key是因為以SSH而非HTTPS模式需要，而不用HTTPS模式是因為那樣要帳號和密碼，而自動化時明文輸入帳號密碼是非常不安全的！

2.建立Github倉庫

推薦選擇私人(private)，即使你備份的是一個網站，但是考慮到如果是公開(public)的，可能讓你的整個網站被盜走。
