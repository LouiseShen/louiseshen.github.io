macOS開發環境搭建——打造mac神兵利器

  • 釋出時間：2019-09-14 08:05:00
  • 1293字數
  • 分類： 好東西
  1. 首頁
  2. 正文

Head Pic: #バーチャルYouTuber さなちゃん - えるめんと's illustrations - pixiv

macOS作為一個兼有Linux的CLI和非常nice的GUI的系統，用來開發最適合了。

編譯器

Java

去官網下載對應mac版本安裝就OK了

php

php是預設就安裝好的

php-fpm啟動就直接在終端敲sudo php-fpm（第一次可能會報錯，但是我這裏無法重現這個問題，所以幫不到你了）

gcc&g++

這兩個也是預設就安裝好的（如果沒有請在終端機直接敲g++，會提示你安裝xcode命令列工具）

IDE

Eclipse

我不能說用eclipse來開發Java究竟是因為好用，還是純粹出於習慣，從JDK1.7一直用的JDK1.10+。

對macOS配置非常ok的，直接去官網下載，然後安裝就行啦，（想想以前在windows還要設定環境變數balabala的，不堪回首的過去）

Visual Studio Code

微軟出品IDE，外掛非常豐富，GUI也非常好看。

官網下載。

透明化可以參考我之前的一篇文章：

[VS Code]VS Code 設定透明背景（支援win/mac）

其他

nginx

有時侯想開發web程式，但是總不能每次都扔生產環境去debug吧。

做一個本地伺服器也是不錯的。

對於mac還是推薦編譯安裝（畢竟比雲伺服器效能強多了，編譯一下就OK了），如果是第一次編譯安裝請使用搜索引擎，這裏大概只有一個步驟（這個東西太囉嗦啦）。

  1. 從官方下載，解壓
  2. 開啟終端，cd到你解壓出來的目錄
  3. ./configure，make，make install
  4. 看下編譯結束資訊，你的nginx程式放到哪裏去了

一般來說，會提示缺少依賴，這時候只需要去再去編譯安裝依賴項即可。

mysql

對於有GUI的系統，mysql都很好安裝的，直接去下載，然後雙擊安裝即可。

這樣做完，就等於有了MNMP環境啦233（macOS + nginx + mysql + php），還有後端Java,php開發。
