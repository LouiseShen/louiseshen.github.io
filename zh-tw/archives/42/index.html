[GFW]v2ray食用指北（重製版）

  • 3379字數
  • 分類： 伺服器技術？不存在的好東西

Head Pic: 【かぐや様は告らせたい】"Love is War" / Illustration by "gomzi" [pixiv]

對於已公開的合法資訊，人民有自由獲取的權利。同時，人民也有言論自由的權利。——v2ray.com

又是一個簡短的口胡教程，大概只能指明一個方向，具體要多用搜索引擎。

介紹

見白話文教程或官網

咳咳都被牆了。

簡單來說就是比ss具有更完善的協議（VMess），更好的效能（見其部落格），更豐富的功能（這個在官網高階篇有）。

但是也有更復雜的配置檔案。

伺服器

首先你需要一個非大陸伺服器，我這裏用的阿里雲輕量咳咳監控一解除安裝穩的一匹。

1c1g 25G 1TB流量 最高30Mbps，才24 CNY ￥，可以說非常實惠了。

安裝

如果你像我一樣懶，那就隨便找個一鍵安裝指令碼之類的來裝就行了。比如這個來自233v2.com的。

一般來說就是這樣：

$ wget https://raw.githubusercontent.com/233boy/v2ray/master/install.sh
$ chmod +x install.sh
$ ./install.sh

當然你也可以用官網的安裝方式。

基礎配置

指令碼上都很明白的，一般傳輸方式用TCP就足夠了,加密方式選擇自動就好了。其他的可以自行了解。

進階配置

websocket + cloudflare CDN

websocket有個好處就是可以使用nginx等伺服器分流，這樣就可以使用CDN服務，當然，在意速度的話，不建議在晚高峰非移動節點使用cloudflare。

服務端配置（示例，id，埠號和ws的path都是可以自定義的）

{
  "log": {
    "access": "/var/log/v2ray/access.log",
    "error": "/var/log/v2ray/error.log",
    "loglevel": "warning"
  },
  "dns": {},
  "stats": {},
  "inbounds": [
    {
      "port": 66666,
      "protocol": "vmess",
      "settings": {
        "clients": [
          {
            "id": "83a3b6c8-ed15-4169-8c14-ae60b3c718d8",
            "alterId": 32
          }
        ]
      },
      "tag": "in-0",
      "streamSettings": {
        "network": "ws",
        "security": "none",
        "wsSettings": {
          "path": "/fuckgfw_websocket"
        }
      },
      "listen": "127.0.0.1"
    }
  ],
  "outbounds": [
    {
      "tag": "direct",
      "protocol": "freedom",
      "settings": {}
    },
    {
      "tag": "blocked",
      "protocol": "blackhole",
      "settings": {}
    }
  ],
  "routing": {
    "domainStrategy": "AsIs",
    "rules": [
      {
        "type": "field",
        "ip": [
          "geoip:private"
        ],
        "outboundTag": "blocked"
      }
    ]
  },
  "policy": {},
  "reverse": {},
  "transport": {}
}

這裏是nginx的配置：

server {
    listen 443 ssl;
    server_name 你使用的域名;

    ssl_certificate 證書路徑;
    ssl_certificate_key 私匙路徑;
    ssl_session_timeout 5m;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_prefer_server_ciphers on;

    root /usr/share/nginx/v2ray;

    location / {
        try_files $uri $uri/ =404;
    }

    location /fuckgfw_websocket { #這裏和wsSettings的path一致。
      proxy_redirect off;
      proxy_pass http://127.0.0.1:66666;//埠號也一致。
      proxy_http_version 1.1;
      proxy_set_header Upgrade $http_upgrade;
      proxy_set_header Connection "upgrade";
      proxy_set_header Host $http_host;
      # Show real IP in v2ray access.log
      proxy_set_header X-Real-IP $remote_addr;
      proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }
}

然後你就可以直接用了，或者用開啟CDN。

客戶端配置

Windows

先下載v2ray-core，然後下載v2ray-N(v2ray的第三方win圖形客戶端)

配置很簡單的就嘻嘻。

iOS

推薦使用Kitsunebi或者i2ray好像美區纔有，要1刀（前者，後者2.99刀）。

配置嘛也很簡單的。

其他平臺

去官網呀。
