[JavaFX]JavaFX中ListView雙擊事件

  • 釋出時間：2019-01-26 15:03:00
  • 917字數
  • 分類：
  1. 首頁
  2. 正文

Head Pic: 【女の子】「miku 魚」/「千夜QYS3」のイラスト [pixiv]

最近在一個專案中用到了ListView這個元件用來實現檔案瀏覽器。

問題

在通常的作業系統中，我們都是雙擊開啟一個目錄的，而JavaFX給的API文件中是這樣給出選擇事件的：

fileManager.getSelectionModel().selectedItemProperty().addListener(
        (ObservableValue<? extends String> ov,String oldVal,String newVal) -> {
    //Your code here.
});

用一個$lamda$表示式來完成，但是這樣的事件是選擇事件，而非咱們要的雙擊。

解決

雙擊自然是要想到滑鼠了，就給ListView監聽滑鼠事件即可。

fileManager.setOnMouseClicked(new EventHandler<MouseEvent>() {
    @Override
    public void handle(MouseEvent event) {
        if(event.getButton() == MouseButton.PRIMARY && event.getClickCount() == 2) {
            TypeName currentItem = fileManager.getSelectionModel().getSelectedItem();
            //Your code here.
        }
    }		
});

其中TypeName是你ListView存放的型別。
