触控板

一开始我并不理解为何能够远离鼠标，因为有一件事情——长距离拖移是一般触控板不适合的，而当我发现mbp触控板内置的但是却不是默认打开的「三指拖移」选项后，我意识到真的可以完全不用鼠标了。（鼠标：？？？）

  • 三指拖移 「系统偏好设置」「辅助功能」「鼠标与触控板」「触控板选项」
  • 三指查词 「系统偏好设置」「触控板」「光标与点按」勾选查找与数据检测器
  • 滚动方向 推荐使用触控板时保留默认滚动方向，也就是自然方向
  • 两指滚动 双指滑动即可代替滚轮，还可以额外的进行横向滚动
  • 四指左右滑动 可以快速在多个全屏幕视窗之间切换
  • 四指上滑 平铺所有打开的窗口

快捷键

  • command + shift + . 在Finder中隐藏/展示隐藏文件
  • command + i 显示简介
  • command + ←和command + → 跳转到行首行末
  • command + ↑ 返回上级文件夹
  • command + space 打开「聚焦」
