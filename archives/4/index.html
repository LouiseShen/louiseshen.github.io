The Keyboard I

Louise
default

Post on Dec 20, 2018
views

My first mechanical keyboard which was not the custom one, was technically broken and the PCB failed. I have no idea how to fix it.

Building a custom mechanical keyboard is a BIG step into the world that is mechanical keyboards.

In this article I’m going to build my first custom mechanical keyboard.

Preparation

Basically, we’ll need to figure out:

  • What size keyboard we need
  • What kind of key switches we need
  • How mucn does all things to pay

Finding the right size

The origin is full-sized, commonly used on pre-built keyboard. However, it just too large for me to use and a lot of keys are totally uncessary foe typing.

Thus, I choose 60% layout for this build. 60% layout is the most common custom keyboard which you can buy. There is no function rows, numpad or navigational cluster and actually it can have arrow keys if you want.

Other components

Because it’s my very first builing, it’s very cheap.

Keyboard case GH60 plastic
Plate GH60 steel
PCB GH60 60% (no bluetooth)
Stabilizers No brand
Switches Kailh Box Gren
Keycaps ABS

Installing

And it’s also my very first soder work. It looks really not good, but IT WORKS!

Final

To be honest, my first custom keyboard looks not great, but it works just fine and that’s enough for me.
